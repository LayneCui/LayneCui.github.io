入行一年半，小小感悟

发表于 2017-07-28

16年，我过的非常闲。
1月到8月在上家公司，只完成了一款APP的上线工作。
8月到11月身在江湖，嘿嘿，这期间的荡气回肠过几天单独开一篇。
11月到新公司，团队兵强马壮，上线项目半个月迭代一个版本，每个版本分到手的任务也只够喝一杯的。
正因为这样，我才有时间去感悟，有时间谈感悟。

###关于工作：
马上年底啦，很快春风又绿江南岸，又是一年跳槽时。对于换工作，相信不少人都有过这样的疑问：有的工作是累一点、忙一点，比如大多数外包公司或者创业型小公司，而有的工作会比较闲一点、轻松一点，比如没有成本压力的大公司，这两种工作到底哪种对我们技术人的成长和发展才会有更大的帮助呢？

我个人的观点是：闲一点，轻松一点的工作环境，更有利于我们提升自己。理由是这样的: 我们都知道，日前发展最迅速的行业就是互联网相关行业了。作为IT从业人员，对于这点估计体会最深刻的就是新技术、新思路的不断涌现了。说目不暇接，日新月异，也毫不为过。我想这也正是IT行业高薪的原因，你需要不断的去发现变化，拥抱变化，甚至去创造变化，才能保证自己不被淘汰。这不是谁都能做到的，薪水永远不会和你的工作努力程度成正比，却一定会和你工作的不可替代性成正比！在这样的大环境下就要求我们最好要有一定的时间来不断的提升自己，跟上节奏，让自己更加独特，变得更加不可替代。一味的埋头敲代码无论对你技术的增长，还是职业的发展都没太大好处。给自己留出思考和学习的时间，太累太忙会让你无暇思考。对于技术你需要有时间归纳和更新，对于个人发展你更需要有时间去思考和涉猎。

###关于面试：
忘记了从什么时候开始了，大概是去年冬季吧，从身边，互联网上，听到关于工作不好找的声音越来越大。这我不否认，原因这里不做细致分析，大概就是
大环境的影响（资本寒冬）、市场供求关系影响（培训班泛滥）、还有就是互联网发展由泡沫严重慢慢恢复到正常稳定的状态。但是这些并不是你找不到工作的理由或者借口。因为不好找并不代表没有工作岗位，打开招聘网站就能看到，多数企业还是有岗位需求的。至于为什么你找不到，你应该多找找自身原因，任何行业，起起伏伏中都会有人被淘汰，更有新鲜血液注入，被淘汰的都是行业竞争中的弱者，没机会进入循环的也是这样。对于那些说投了一个月甚至几个月简历都没有面试邀请，根本没机会的，我对你们是佩服的。你会重复投简历一个月，没有一个面试邀请，却不回头想想到底哪个环节出了问题？额，好吧你赢了。
前边说过，今年换了一次工作，所以很幸运的又经历了一次找工作。
我的战绩：
9月份整好简历，用了2天时间在拉勾用2个号，投了30份简历，有8家面试邀请，1家电话面试邀请（其中有2家是Boss直聘约的）推了其中2家面试，第1天面了3家拿到1个offer，第2天面了3家拿到2个offer，晚上电话面试由于信号原因没面成。先说明，我技术很一般，从毕业开始找工作就没觉得工作有多难找。因为我每个环节都认真对待了，都用心了，所以不要看工作多难找，不要听周围有多少人找不到工作。做好该做的，工作就摆在你眼前。
我的经验：
简历很重要！简历很重要！简历很重要！三遍，原因你懂得。

  • 1.找模板：
    有好的简历模板，起到的效果是潜移默化的。模板哪来？平时积累。技术简历一定要简单明了。来看我使用的一个模板：
  • 2.写简历：
    简单清晰，不该有的别有，什么身高体重肺活量啊，什么老家是哪门牌号啊，没人关心你这些！动脑子想想企业招人想知道关于你的什么。简历篇幅最好控制在两张A4纸以内，所以明确什么该写什么不该写。
    写简历是有窍门的，说说我写简历的经验。
    硬指标我们不说，学历，年龄，学习经历等等我们改变不了什么。但是简历上另外两块：“职业技能”和“工作经历”是最重要的也是值得花费精力好好填写的，
    前者是你的技术体现，后者是你的工作能力和熟悉的业务在之前工作项目中扮演的角色。
    职业技能填写我是这样做的：首先打开拉勾网或者Boss直聘，筛选出符合你条件的正在招聘企业，看看他们的职位描述和任职要求吧，对！知道他们需要什么样的简历了吧？如果你说他们的任职要求你都无法达到，甚至都看不懂，那没办法了···
    如果你对某一家企业特别看重，特别想去这家公司工作，那就给他单独做一份简历吧。谁让你爱它呢，爱就要学会付出。不要想着一份简历搞定所有企业，你这么懒还说什么爱。 知道他的行业，明白他的需要，从自己曾经写过的项目中找出和这家类型最契合的项目，写在你的项目经历上。你的简历要做的就是让他们HR看到之后惊叹一声！“终于找到你，还好没放弃···”约你面试那不是分分钟的事嘛！
    至于约到之后面试能否成功就是你下一个环节需要做的了。为了不让下一个环节的自己坐马桶都有猝死的危险，此招虽好但是也不要过于难为了自己，还浪费彼此的时间。
  • 3.准备面试：
    准备面试阶段也是最苦的，因为你可能为了拿到面试邀请，在简历上吹了小小的牛逼。这时候就是为自己吹的牛逼负责的时候。把你所有版本的简历上职业技能一栏一条条的去实现，哪怕没有时间实践但是也要保证理论无虞。能不能短时间内磨刀霍霍，就考验你知识储备库够不够用了，关于知识储备我下边说。
  • 4.面试：
    面试过程其实是最简单的过程了，该答题答题，该提问提问，不要卑躬屈膝，也不要目中无人，不卑不亢！我想留下但是也不是非这不可。注意不要让面试官觉得你稳定性太差，这点最主要体现是在面试官问你为啥离职。毕竟企业招一个人进来也挺不容易的，所以要妥善回答。不要觉得是你开除了上家公司，面试官会觉得你很牛逼。
    最后补充一点：
    简历上的薪水能多写不少写，简历上的薪水取决于你对自己的信心和对自己的定位。告诉你，业内人才分布按照薪水划分呈一个金字塔形，你在金字塔的位置越靠上和你竞争的人就越少。有动力了吗？还有互联网企业其实大多数不差钱差的就是有能力的人。所以薪水多要一些吧，多出的薪水浪迹江湖，诗和远方···

###关于业余时间：
业余时间，我觉得最重要的就是构建自己的知识储备库并且精准的定位自己道自己想要晋阶需要做什么。知识储备库，是你强大自信的支撑，是你提高自己的营养来源，更你平时装逼的谈资 TAT~~；定位好自己主动去晋阶会从根本上更快的提高自己。

  • 知识储备库：
    首先要有信息来源，最好还是实时的，新鲜的。因为这样能保证有什么新技术，你能第一时间知道。新技术能够解放劳动力，掌握了甚至一天完成一周的任务这样你就更闲了。当然这里知识打个比方。不过真的要拥抱改变，这样你才能不断提升自己的竞争力。
    要时刻关注着业内的动态，不要闭目塞听，大牛或者大企业的一些优秀开源项目对你真的很有价值。
    无论是社区，论坛，公众号，APP等等，只要有价值的统统拿来去粗取精，找到最适合自己的一个或者几个，也不要太多。零碎时间，上班下班拿出手机来刷，从这里获取业内的动态，老鸟们的心得等等你所需要的一切信息，这里全都有。这些信息对你的价值是无可估量的。上班没事时去论坛看，去社区找。
    有了信息来源，怎样创建知识库呢？因为信息来自不同的地方，所以我们要找到适合的APP来多端多来源的同步归纳到一款APP里边那就太好了，“印象笔记”你值得拥有，大多数APP里都有分享功能，当遇到值得收藏的文章时可以直接分享到印象笔记，印象笔记可以创建不同分类的笔记本，方便你整理。值得一说的是印象笔记还有谷歌插件，看到有价值的网页也可以一键收藏到印象笔记中，6不6？所以我建议，我们是做IT的，要体验各个APP，去玩各种各样的APP，让他们为我们所用。所以买个好点的手机吧，好好利用它，好好玩它。
    注意的是，遇到好文章不需要立刻处理，知识储备库嘛，存起来，先不消化，做好分类，用时能找到就好。当然有些文章你需要尽快处理，那就放到一个指定收藏夹这个收藏夹里的东西，你有时间就要去处理一下，不要越积越多，到时候自己都懒得看了。
  • 定位晋级：
    上边第一条主要讲的是开阔视野，收藏轮子的功夫。而作为一个有追求的程序员TOT～～（忍住别笑），我们不能总用别人的轮子，自己也要有造轮子的技术在手啊。着急时拿来用，上边方法能给你节省时间，解决后顾之忧，因为经过了自己的筛选，比着急用时满世界去找要好得多还节省时间。节省下来的时间干嘛呢？我们可以自己造轮子啊！造好轮子让别人去用，想想就有成就感！
    想要晋阶怎么做呢？
    首先准确定位自己，然后明确晋级目标。比如我现在给自己的定位是中级工程师偏上一点水平，我想晋级到高级水平。首先，明确高级的职能：有搭建框架能力，掌握设计模式，面向对象编程，封装继承多态熟用，自定义view能力，新技术（各种优秀开源库，新架构，新思想）了解使用源码阅读按需修改能力，等等，了解自己现在状况（以上啥也不会）。
    高级的职能 - 自己现在状况 = 你需要做的。
    知道自己要做啥了，那就干吧！买书、看书，练习！看优秀部落格，练习！看优秀开源项目，练习！每个行业都一样，如果你想成为该行业的高手，刻意练习，是不二法门。是刻意的哦，有目的性的哦！

就这样，有了储备，工作就会得心应手；有了练习，你就会在原有基础上提高！让自己闲吧，闲下来的时间来让自己更闲。
